# command-not-found
![node-current](https://img.shields.io/node/v/command-not-found)


a cli tool that let you found the missing command.

## Install

```bash
npm install -g command-not-found
```

## Quick Start

```bash
# search nc
cnf nc

# show help
cnf -h
```
